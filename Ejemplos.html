📝 Memoria de Cálculo: Método de Gauss-Seidel

Resolución detallada y convergencia de errores

1. Definición del Sistema y Despejes

Para el sistema de ecuaciones propuesto, realizamos los despejes de las variables de la diagonal principal:

$$x_1^{(k+1)} = \frac{9 + 2x_2^{(k)} + x_3^{(k)}}{10}$$ $$x_2^{(k+1)} = \frac{7 + 2x_1^{(k+1)} + x_3^{(k)}}{10}$$ $$x_3^{(k+1)} = \frac{6 + x_1^{(k+1)} + x_2^{(k+1)}}{5}$$

2. Primera Iteración (k=1)

Partiendo de valores iniciales x1=0, x2=0, x3=0:

Cálculo de variables:

$$x_1^{(1)} = \frac{9 + 2(0) + 0}{10} = 0.9000$$ $$x_2^{(1)} = \frac{7 + 2(0.9000) + 0}{10} = 0.8800$$ $$x_3^{(1)} = \frac{6 + 0.9000 + 0.8800}{5} = 1.5560$$

Cálculo de errores relativos porcentuales:

$$|E_{a,1}| = \left| \frac{0.9000 - 0}{0.9000} \right| \times 100\% = 100\%$$ $$|E_{a,2}| = \left| \frac{0.8800 - 0}{0.8800} \right| \times 100\% = 100\%$$ $$|E_{a,3}| = \left| \frac{1.5560 - 0}{1.5560} \right| \times 100\% = 100\%$$

3. Segunda Iteración (k=2)

Sustituimos los valores obtenidos: x1=0.9, x2=0.88, x3=1.556:

Cálculo de variables:

$$x_1^{(2)} = \frac{9 + 2(0.8800) + 1.5560}{10} = 1.2316$$ $$x_2^{(2)} = \frac{7 + 2(1.2316) + 1.5560}{10} = 1.1019$$ $$x_3^{(2)} = \frac{6 + 1.2316 + 1.1019}{5} = 1.6667$$

Evolución del error (Convergencia):

$$|E_{a,1}| = \left| \frac{1.2316 - 0.9000}{1.2316} \right| \times 100\% = 26.92\%$$ $$|E_{a,2}| = \left| \frac{1.1019 - 0.8800}{1.1019} \right| \times 100\% = 20.14\%$$ $$|E_{a,3}| = \left| \frac{1.6667 - 1.5560}{1.6667} \right| \times 100\% = 6.64\%$$
